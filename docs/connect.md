# [EN] 🖥️ Method via properties «osu!»

1. Create a **«osu!»** shortcut.
2. Right-Click it.
3. Find the **«Properties»** line.
4. And find the **«Target»** line.
5. Enter the following at the very end: ` -devserver osutraktor.ru`

* * *

# [RU] 🖥️ Метод через свойство ярлыка «osu!»

1. Создайте ярлык **«osu!»**
2. Правый клик по нему.
3. Выберите пункт **«Свойство»**
4. Найдите строку **«Объект»**
5. И введите в самый конец, следующее: ` -devserver osutraktor.ru`
